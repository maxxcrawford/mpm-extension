Mozilla Privacy Manager

What does this extension do?

This extension helps users manage their privacy by automating tasks for them. These tasks include updating social network privacy settings and filling out CCPA request forms.

What's next?

  • Increase site actions
  • Support GDPR request forms
  • Promote other Firefox services like Monitor and VPN
  • Handle multiple autofill profiles
